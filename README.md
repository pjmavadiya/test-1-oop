# test-1-oop
Write a Python class named Circle constructed by a radius and two methods which will compute the area and the perimeter of a circle.

## Usage
```
python circle.py
Radius: 5

Area: 78.539816
Perimeter: : 31.415927


Provide radius as a command line argument...
```

```
python circle.py 10

Radius: 10
Area: 314.159265
Perimeter: : 62.831853
```
